Phonegap Demonstration using jQurey mobile

This application demonstrates the combination of Phonegap and jQuery mobile to create an HTML5 based mobile application with responsive design. This app mostly provides examples of my own work, rather than explainging jQuery Mobile. To understand jQuery Mobile, see the jQuery Mobile documentation website.
